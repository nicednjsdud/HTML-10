CSS
====

## CSS (Cascading Style Sheets)

* HTML 문서를 시각적으로 꾸미는 역할
* 한번 작성에 여러 HTML 페이지에서 재사용 가능
* CSS 파일만 변경해도 해당 CSS를 적용한 모든 웹 페이지 요소가 자동으로 업데이트됨.

## CSS 적용하기

### 간단하게 스타일을 적용하는 방법

```css
    <style>
        <!-- CSS 코드 -->
    </style> 태그 사용
```
