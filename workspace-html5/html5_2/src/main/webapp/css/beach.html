경포 해수욕장

강원도 강릉시

기본 정보

경포 해수욕장은 강원도 강릉시 안현동에 위치한 해수욕장이다. 모래사장의 총면적은 144,000m², 길이는 1.8 km, 폭은 80m이다. 경포해수욕장은 강원도 강릉시 강문동·안현동 일대에 있는 동해안 최대의 해수욕장이다. 피서철에는 하루 평균 50만명의 피서인파가 몰리기도 한다.

  • 홈페이지 : https://www.gn.go.kr
  • 주소 : 강원도 강릉시 안현동
  • 애완견 : 동반가능, 시각장애인 편의시설
  • 주차 : 장애인 주차장 있음
  • 유모차 : 대여가능(2대, 경표해변 관리본부)
